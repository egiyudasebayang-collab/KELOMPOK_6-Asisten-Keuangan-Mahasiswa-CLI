# Kelompok6-Asisten-Keuangan-Mahasiswa
# 💰 Asisten Keuangan Mahasiswa CLI

Program berbasis terminal (CLI) yang dirancang untuk membantu mahasiswa dalam mengelola keuangan sehari-hari secara sederhana dan praktis.

---

## 📌 Deskripsi

Asisten Keuangan Mahasiswa CLI adalah aplikasi berbasis bahasa C yang menyediakan beberapa fitur perhitungan keuangan dasar, seperti menghitung diskon, membagi tagihan, menghitung bunga tabungan, dan memantau sisa uang harian.

Program ini dibuat sebagai bagian dari tugas kelompok untuk melatih:
- Pemrograman dasar C
- Modularisasi fungsi
- Kolaborasi tim menggunakan Git

---

## ⚙️ Fitur Utama

1. **Kalkulator Diskon**
   - Menghitung harga akhir setelah diskon

2. **Split Bill**
   - Membagi total tagihan ke beberapa orang

3. **Bunga Tabungan Sederhana**
   - Menghitung saldo akhir berdasarkan bunga

4. **Sisa Uang Harian**
   - Mengetahui sisa uang setelah pengeluaran
